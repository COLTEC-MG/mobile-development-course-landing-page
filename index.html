O curso

Voltado para quem deseja ingressar no mercado que mais cresce na área de Tecnologia da informação, o Curso Técnico de Desenvolvimento de Sistemas com ênfase em programação para dispositivos móveis tem como objetivo formar profissionais de grande competência para atuar no desenvolvimento de aplicativos móveis nas plataformas Android, iOS e Web. O profissional que finalizar seu curso será certificado como técnico em desenvolvimento de sistemas por uma das instituições de maior prestígio do país.

Objetivos

Ao finalizar o curso, você estará apto a desenvolver aplicativos e sistemas correlatos, bem como utilizar fluentemente ambientes de desenvolvimento, sistemas operacionais e gerência de base de dados que viabilize a estruturação, segurança e a consequente manutenção ou atualização deste tipo de software.

Você irá apreender a...

  • Desenvolver aplicativos móveis para iOS e Android utilizando as ferramentas e tecnologias mais atuais do mercado. seguindo as especificações e paradigmas da lógica de programação e das linguagens de programação;
  • Utilizar ambientes integrais de desenvolvimento de sistemas, sistemas operacionais e os principais sistemas gerenciadores de banco de dados existentes no mercado;
  • Realizar tarefas envolvendo testes automatizados, correção e manutenção de programas de computador;

Módulo 1

Introdução a Programação

Formar o aluno nos conceitos básicos e intermediários de programação de computadores e lógica computacional. Ao final desse módulo o aluno estará apto a desenvolver programas em C e compreenderá a estrutura de harware e software de sistemas de pequeno e médio porte.

Módulo 2

Programação de Computadores

Formar o aluno nos conceitos de programação de computadores e estrutura de dados visando capacitá-lo para o desenvolvimento de programas estruturados. Ao final desse módulo o aluno poderá desenvolver programas em Java utilizando a programação orientada a objetos em conjunto com Banco de Dados.

Módulo 3

Programação para Dispositivos Móveis

Desenvolver aplicativos para dispositivos móveis nas plataformas Android e iOS, resolvendo problemas reais para o mercado de trabalho e centros de pesquisa e ensino. Ao final desse módulo o aluno estará apto a desenvolver aplicativos reais em ambientes para Android e iOS e obter o diploma de técnico em informática.
